To run this sample project, you need to register to https://newsapi.org/ and put the api key in the project.

A sample of how get the api and show the result to the list is already given.

You are asked to improve the page by the following tasks.
  • Improve the view by showing the loading state when loading the data, empty state when the data is empty, and error and retry state when the data is error.

  • Currently the page is showing only up to 20 data of the first page. You have to add capability to Load More if the user go to the bottom of the list.

  • Show the Slider Banner in the top of the list. It shows up to 5 of headline news data (See documentation from the newsapi.org!).

    User can slide the slider to the left/right. The slider has the page indicator in the bottom and periodically can turn to next page every 5 seconds.

    Below is illustration of the slider


  • Format the date! Instead of 2009-01-10T20:45:00Z, you have to put the date to 10 January 2009. The date should aligned with the user's local device.

  • Instead of showing the date to each item, You are asked to group the news by date.
    For example:
    10 January 2019
    [News 1]
    [News 2]
    9 January 2019
    [News 3]
    [News 4]
    [News 5]

  • Add Search capability to allow the user to search the news by the keyword he/she wants.

  • Add offline capability by storing the news to the database, so the news can be shown to user even he/she is not online.
