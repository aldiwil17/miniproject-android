Given a graph view that represent a tree (there is no cyclic), a rootNode, and an integer of target distance, you have to traverse the graph and give the color to each nodes in the graph with the following criteria:
  • Root Node is colored purple.
  • Nodes with the distance are less than the target distance are colored green.
  • Nodes with the distance are equal to the target distance are colored orange.
  • Other Nodes outside the above criteria are blue.
*Distance is the number of distinct edges from a node to the rootNode.

Example

Input

Graph =

RootNode = Node 1
Target Distance = 2

Output

Explanation

Node 1 is root, so it is colored purple
Node 2 has distance 1 to Node 1, so it is colored green
Node 3 has distance 1 to Node 1, so it is colored green
Node 4 has distance 2 to Node 1, so it is colored orange
Node 5 has distance 2 to Node 1, so it is colored orange
Node 6 has distance 2 to Node 1, so it is colored orange
Node 7 has distance 2 to Node 1, so it is colored orange
